Om oss

Ingrediensen ger receptet är en enkel tjänst för att hitta recept genom sökning på rättens namn.

Mål: hjälpa matglada personer 25–50 år att laga god mat utan krångel.

Teknik: Semantisk HTML, responsiv CSS (mobile–desktop), och två JavaScript-sidor som hämtar data från öppet API.
